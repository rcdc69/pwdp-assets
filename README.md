# pwdp-assets

Shared member / accreditation logos for PWDP estate-agent websites.

Each logo is `members/<key>.png` and `members/<key>.webp` (200x70 canvas).
Served free via the jsDelivr CDN:

```
https://cdn.jsdelivr.net/gh/rcdc69/pwdp-assets@main/members/<key>.webp
```

Update by pushing a new/replacement file. jsDelivr caches `@main` for a while;
for an immediate refresh reference a tag/commit (e.g. `@v2`) or purge via
`https://purge.jsdelivr.net/gh/rcdc69/pwdp-assets@main/members/<key>.webp`.

Used by the `new-ea-site` skill and the PWDP estate-agent sites (e.g. Carvels).
